‚öôÔ∏è How to Run
================
To run the project, follow these steps:
1. **Setup**: Install the required dependencies by running `pip install -r requirements.txt`
2. **Environment**: Set the following environment variables:
	* `OWM_API_KEY`: Your OpenWeatherMap API key
	* `TWILIO_ACCOUNT_SID`: Your Twilio account SID
	* `TWILIO_AUTH_TOKEN`: Your Twilio authentication token
3. **Build**: Build the project by running `python main.py`
4. **Deploy**: The project can be deployed on any platform that supports Python 3.x, such as a cloud provider or a local server
